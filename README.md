# string-writer
Скрипт для демонстрации кода

*Необходим Python 10*

Запускать так:
```bash
python app.py [OPTIONS] <full filename>
```

Доступные опции:
* -h          - Вывод справки
* -s <1-100>  - скорость вывода текста на консоль
